# Zero Knowledge Proofs

An interactive guide to zero-knowledge proofs — covering theory, protocols, and demos.

**Live site:** https://duoxehyon.github.io/zk-proofs/

> Work in progress. Content and pages are being added incrementally.

## Pages (so far)

- **Introduction** (`index.html`)
- **Graph Coloring** (`chapters/graph-coloring.html`)
- **Sudoku ZKP** (`zkp.html`)
- **Ali Baba Cave** (`chapters/alibaba.html`)

## Running locally

Just open `index.html` in a browser. No build step or server required.
